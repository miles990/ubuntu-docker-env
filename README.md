# Deploy ubuntu 14.04 virtual machine via vagrant and install Docker via script

#### Required
- Virtualbox
- Vagrant

#### Deploy ubuntu vm
- Installation
`
vagrant up
`

- Connect
`
vagrant ssh
`

#### Install docker on vm
- Installation
`
./install_docker.sh
`

#### Install docker web ui
- Installation
`
./install_shipyard

`
- web ui [http://localhost:8080](http://localhost:8080)
